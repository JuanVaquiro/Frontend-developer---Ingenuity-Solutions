## Run Locally

Clone the project

```bash
  git clone https://github.com/JuanVaquiro/Frontend-developer-Ingenuity-Solutions.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies
Frontend-developer with npm

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
